# ZEN大学 奨学金申請プロジェクト 概要

> **提出日**: 2035年6月10日  
> **申請者**: momo  
> **目的**: 韓国自主研修および事業立ち上げに対する奨学金申請

---

## 🎯 プロジェクトの概要

祖父母の家を活用し、余市町の学生が挑戦・活動できる場を創造する。  
任意団体としてスモールスタートし、在学中に **一般社団法人** と **合同会社** のハイブリッドモデルへ育てる。  
本奨学金は、この企業へ向けた **韓国の自主研修** へ充てる予定。

## 📅 4ヶ年ロードマップ

### 1年生「世界で学び、地元で試し、発信で磨く。起業前の実験の一年にする！」

**大学外での活動計画:**

| 期限 | 活動内容 |
|------|---------|
| 4月〜6月 | 運転免許取得 / 既存のコミュニティ研究 |
| 7月〜9月 | 簿記3級取得 |
| 10月〜12月 | 韓国自主研修 / FP3級取得 |
| 1月〜3月 | シリコンバレー留学 / 学びを発信 |

### 2年生「収入の柱を立てながら実践的に『お金を稼ぐこと』を学ぶ！」

- FP2級・簿記2級取得
- SNS広告収入、SNSアフィリエイト、ココナラ、塾、ノート記事、会計バイトなどの勉強ができる仕事で **6本柱** で生計を立てる
- 会計のバイトもしくはインターン入る、任意団体として立ち上げて学生団体にする
- 単発イベントまで繋げる、SNS1000人目指す、既存のコミュニティリサーチ

### 3年生「起業するために小さく事業をスタート」

- 余市町の中学生・高校生向けオンライン進路相談
- 「余市から世界へ」発信プロジェクト
- ワイナリーやカフェ、学生をインタビューして投稿
- 余市町の子ども向け英語・留学体験ワークショップ（出張型）
- 教育格差調査レポートの発信
- 余市町の高校生向けビジネスアイデアコンテスト（オンライン開催）
- 塾講師バイトで教育現場を経験する
- マーケティングか営業のインターンに入る

### 4年生「法人化・収益化し事業を拡大！」

- 余市から世界へプロジェクトを収益化
- ワークショップを収益化
- 任意団体から一般社団法人やNPO法人にする
